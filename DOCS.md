## 💬 Getting Help

### Resources in Order of Priority

1. **Search these docs** (quick answer)
   - Use Ctrl+F to search on GitHub
   - Check troubleshooting sections first

2. **Check GitHub Issues** (common problems)
   - Search existing issues
   - May find solution to your exact problem

3. **Refer to QUICK_START.md** (setup help)
   - Most issues are covered here

4. **Open a GitHub Issue** (new problem)
   - Include OS, Java version, error message
   - Describe steps to reproduce
   - Mention which doc you've read

5. **Ask for Help** (discussions)
   - Use GitHub Discussions tab
   - Community may help faster
